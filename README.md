# Kims dotfiles and MacOS setup

[Terminal theme](http://i.imgur.com/8psgBag.png)

# Install 
```
git clone https://github.com/kimsyversen/dotfiles && cd dotfiles && bash brew.sh && bash bootstrap.sh --force && bash macos.sh
```
